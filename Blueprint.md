# 기획
기본 콘셉트 - 미로찾기 
> 미로 타일이 섞여 있어 미로의 형태를 알아보기 힘들다

> 미로자체는 고정되어있지만 보이는 화면만 섞여있어 보이는것과 상호작용이 다르다

> 슬라이드 퍼즐을 통해 미로타일을 재배치하여 미로를 파악해야한다

> 출구가 있는 타일은 슬라이드 퍼즐을 맞추기 전까지 숨겨져 있다

스테이지 구성
1. 골목\
 특별한 상호작용 없이 출구를 찾는 스테이지
2. 숲\
 통나무를 얻어 강타일을 지나가야하는 스테이지
3. 화산 \
 피해를 입는 타일을 최소로 지나가야하는 스테이지

## 화면 흐름 예상
### 시작 화면
게임 타이틀 - Noname \
게임 시작\
게임 정보\
게임 종료 - 진짜 종료\
4개로 구성, 업다운 키로 이동, 선택 시 씬 변경
### 게임 정보 씬
맵 타일 이동 설명 - weadzx \
플래이어 이동 설명 - 방향키\
강 타일은 통나무가 있어야 지나갈 수 있습니다 \
불타는 바닥은 3번만 지나갈 수 있습니다
p키를 눌러 돌아가기

### 게임 시작 씬
스테이지 텍스트 
1. 골목
2. 숲
3. 화산 

클리어 진행도 확인 후 2,3스테이지 회색 처리 \
진입 불가능한 스테이지는 엔터에 반응 없음 \
업다운 키로 이동, 상호작용 가능한 맵 선택 시 씬 변경

### 스테이지 씬
미로 출력 및 오브젝트, 플래이어 표시 \
스테이지에 필요한 **자원 상황** 표시 \
p키 입력 시 **정지 팝업**( 게임 계속, 시작 화면, 게임 종료) \
맵타일 퍼즐 완성 시 숨겨진 타일 공개 \
출구 도달시 **클리어 팝업** (엔터를 눌러 게임 시작 씬으로)

### 미로 화면 구성
세로 30, 가로 49 \
자원 표시칸 좌상단 \
'일시 정지 : P' - 우상단 표시\
조작법 약식 마지막 세로줄


## 구현 할 오브젝트 
1. 플레이어 - P
2. 이동 가능 타일 - 백색(골목), 숲(연두색), 화산(검은색) 
3. 벽 타일 - 회색(골목), 숲(초록색), 화산(붉은색)
4. 강타일 - 파란색(통나무 불가능), 하늘색(통나무 가능), 연한 갈색(통나무 사용) \
통나무 없이 이동 불가능 통나무가 있을 때 벽이 아니면 이동 및 타일 변경 + 통나무 소모
5. 통나무 - 짙은 갈색\
이동 시 획득
6. 불타는 바닥 - 주황색 \
이동 가능 상호작용 시 열기 카운트 증가

### 화면 관리
스크린 매니저가 신 정보를 받아서 신에 맞는 화면 출력 \
보여지는 모든 것에 대한 책임\
맵 매니저가 미로와 오브젝트 위치 상황 관리 \
